# Changelog
All notable changes to this package are documented in this file.

## [1.0.0] - 2026-02-23

### Added

- First public package release of Window Fracture.
- Runtime assets moved to package runtime folders (`Scripts`, `Resources`, `Prefabs`).
- MathNet binaries packaged under `Runtime/Plugins/MathNet`.
- Importable Package Manager sample at `Samples`.
- Package metadata, documentation, and third-party notices completed for publishing.
